# TMDb Movie Rating Analysis
Exploratory data analysis of the TMDb 10,000 movies dataset using NumPy and pandas. Cleans movie metadata and investigates how inflation-adjusted budget, runtime, and genre relate to TMDb user ratings via summary statistics and visualizations in a Jupyter Notebook.

### About this project

This is the first Jupyter Notebook–based data analysis project I’ve shared on GitHub. It was my introduction to working end-to-end with NumPy, pandas, and exploratory data analysis on a real dataset. I’m intentionally leaving this project as-is over time so future work in my portfolio can show how my skills and analysis style evolve as I learn more. I look forward to creating more engaging visuals and working with more complex/meaningful datasets.
